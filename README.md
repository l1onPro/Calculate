# Simple Calculate
Простой калькулятор для Android 5.0 и выше версий. Умеет вычислять простые арифметические операции, такие как: сложить, умножить, разделить, вычесть. 

# Getting Started
Копируйте проект и откройте при помощи Android Studio на своем компьютере. Чтобы протестировать, можно подсоеденить свой телефон по примеру: https://metanit.com/java/android/1.2.php и протестировать на нем.
